# Playback Config                                                             
   
  Authentication page for Playback for Spotify — a 
  Spotify client for Pebble smartwatches.                   
                                                                                
  ## What is this?                                          

  This page handles Spotify OAuth login using the Authorization Code flow with  
  PKCE. It runs on GitHub Pages and connects your Spotify account to the
  Playback app on your Pebble watch.                                            
                                                            
  ## Setup

  You need your own Spotify Client ID to use Playback. See the 
  [Setup             Guide](https://alex523ap.github.io/playback_config/setup.html) for
  step-by-step instructions.                                                    
                                                            
  ## Features

  - Spotify playback control (play, pause, skip, volume)
  - Browse playlists, artists, albums, and liked songs
  - Album art on the watch                                                      
  - Like/unlike tracks                                                          
  - Works on Pebble Time, Pebble 2, and Pebble Time Round                       
  - Spotify Premium required for playback control
